  • The Cams API was created to provide a unified, simple, and abstracted way to interface with the CAMS Enterprise Database by Three Rivers Systems. It is a centralized database that provides an end to end way to manage all the different facets of managing a higher education institution. However, to write custom solutions to interface with it, is a somewhat troublesome experience. That, combined with our older codebases, a new unified portal, and the possibilties of having to interface with third party vendors allowed us to develop a RESTful API system that all of our systems (and third parties) could easily and sanely create, read, update, and delete data.
